# +aura Red Mode MVP

A mobile-first prototype for a social music radar app.

## What is included
- Landing page
- Sign up / login with localStorage
- Onboarding profile flow
- Spotify demo connect step
- Home screen with Red Mode UI
- Radar / map screen
- Signals + chat shell
- Profile page
- Express server for static hosting

## Run locally
```bash
npm install
npm start
```

Then open:
- http://localhost:3000
